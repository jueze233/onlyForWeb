# BanG Dream! 剧情文本转换器 - 功能全览

这是一个专为 BanG Dream! (Bestdori) 社区设计的剧情文本转换工具，可以将普通对话文本转换为 Bestdori 平台可用的 JSON 格式。

通过网页来访问这个工具：
[https://bestdori-webtextconverter.zeabur.app/](https://bestdori-webtextconverter.zeabur.app/)
